# FAQ Chatbot

An AI-powered FAQ Chatbot built using **Python**, **NLTK**, and **Tkinter**. The chatbot provides quick and accurate answers to frequently asked questions related to courses, admissions, fees, placements, and other academic information through an interactive graphical user interface.

## Features

- Interactive GUI using Tkinter
- Natural Language Processing with NLTK
- Pattern-based question answering
- Scrollable chat window
- Easy-to-understand and customizable code
- Lightweight desktop application

## Technologies Used

- Python
- NLTK (Natural Language Toolkit)
- Tkinter
